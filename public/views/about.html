About TICKSTER

TICKSTER is a social movie site for discovering new movies, learning about movies and meeting other people who have similar taste in movies with movie data fetched from the Rotten Tomatoes API. It allows users to create profile, search, like and review movies and follow other users. It also provides you the showtimes for the movies in your area from the TMS API and lets you book them on Fandango.
It is developed in MEAN (MongoDb, Express, Angular, Nodejs) stack and is fast and highly responsive. It implements infinite/dynamic scrolling, so you get your data as you need!

Feature List:
  • Movies Coming Soon: The homepage displays the upcoming movies tickr in a responsive carousal.
  • Movies In theaters: The homepage shows the movies in theatres in a photo grid which on hover displays the movie details. It is implemented with INFINITE/dynamic SCROLLING, so there are no pages. As you scroll down, more content is loaded
  • Search: Search for a movie in the search box. Resuts displayed as a table which is also dynamic/infinite scroll enabled.
  • Movie Details: The Movie Details page shows movie poster, Title, Release Date, Cast, Plot, Genre, Rating, Audience rating and Critic's Rating. This page is displayed if a user clicks on a particular movie.
  • Like/Unlike Movie: You can like/unlike the movie by clicking the like icon after logging in. Its like creating a BookMark for a movie
  • Review: The user and rotten tomatoes reviews are combined into a single section. If you are logged in you can post your own reviews.
  • Showtimes: User can search for movie screening in his nearby area by typing in Zip code and Date. Data is fetched and you can proceed with booking on Fandango.
  • Profile: The user profile page displays profile picture, email, about user along with his favourite movies and the users he/she follows.
  • Follow User: A user can click on a username under reviews and see his profile and follow that user. Once followed, a link to this user is available under followed users in the profile page.
